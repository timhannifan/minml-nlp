# minml-nlp
A starter kit for NLP projects


### Requirements
- Python 3.6
- Pipenv

To install package requirements from pipenv:
```
cd minml-toolkit
pipenv install
```

### Usage
To run a preconfigured topic modeling example:
```
python main.py
```


To run a custom configuration:
```
python main.py '/path/to/my_config.yaml'
```
